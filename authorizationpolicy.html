Why do I need to authorize this site?

The YouTube Data API requires users to authorize sites to use their account for some functions. The search feature on this site is one of those functions. YouTube may require this for several reasons, user security and monitoring use of their API being some of the possibilities.

Manage my account? What does this mean?

The YouTube Data API has many features that one can use with their account, such as uploading videos, doing resumable uploads and creating playlists. However, these are only possibilities, if a site were created to do these functions. AMVgenerator is not able to change your account settings, access your information, or upload videos.
